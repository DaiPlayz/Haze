  <p><strong>Haze</strong> is a high-performance Roblox script built by ScriptIsFocus. It’s designed for users who want stability and advanced functionality.</p>

  <p>Haze <strong>requires advanced exploit functions</strong>, such as:</p>
  <div align="center">
    <p>• <code>hookmetamethod</code></p>
     <p>• <code>checkcaller</code></p>
    <p>• other low-level environment access</p>
  </div>

  <p><strong>Unsupported Executors</strong></p>
  <div align="center">
    <p>• <strong>Xeno</strong></p>
    <p>• <strong>Solara</strong></p>
  </div>

  <p>Most modern, fully-featured executors should work without issues.</p>
  <p>If your executor lacks required functions Haze will not run correctly.</p>

  <h3>Loadstring</h3>
  <pre><code>loadstring(game:HttpGet("https://raw.githubusercontent.com/7Smoker/Haze/refs/heads/main/installer.lua"))()</code></pre>

  <h3>How to Use</h3>
  <div align="center">
    <p>1 - Copy the loadstring above</p>
    <p>2 - Paste it into your executor</p>
    <p>3 - Execute it — on first run Haze will automatically download all required files</p>
  </div>

  <p>After that, you’re set.</p>
